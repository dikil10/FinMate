# FinMate - 个人智能金融助理系统

> 🎯 一个能自行抓取数据、深度分析、给出最优投资建议的智能金融系统。

## 🏗️ 项目目标
- **短期**：搭建个人金融数据中台，实现数据自动抓取与存储
- **长期**：打造集数据管家与投资顾问于一体的智能系统
- **求职定位**：金融数据分析 / 大数据开发 / 量化研究

## 🛠️ 技术栈
| 层级 | 技术 |
|:---|:---|
| 数据抓取 | Python + AkShare |
| 数据存储 | MySQL + Hadoop HDFS |
| 数据分析 | Pandas + Matplotlib |
| 安全测试 | Kali Linux + Nmap |
| 版本管理 | Git + GitHub |
| 未来扩展 | StarRocks + Streamlit + NLP |
